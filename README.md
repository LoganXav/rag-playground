# Modular RAG Pipeline

A collection of small, modular projects exploring **AI pipelines**.  
Each module is self-contained and builds toward practical **retrieval-augmented generation (RAG)** systems.

## Modules

1. **Ingestion** — extract clean text + metadata from PDFs and output chunks.
2. **Embeddings** — convert chunks into embeddings and store in a vector DB.
3. **Retriever** — query documents and return top-k results with references.
4. **Chat Completions** — LLM chat with document context and page references.
5. **Interactive Editor** — frontend to refine and export generated content.
